Custom Select jQuery Plugin Examples:

Github Page

Examples Output Code
Original -
Default (only layout change) JavaScript:

$('#select_1').customselect();
Animated and with page redirection JavaScript:

$('#select_2').customselect({
	animated : true,
	url	: 'index.html?locale='
});
Animated and with callback JavaScript:

$('#select_3').customselect({
	animated : true,
	callback : function() {
		var msg = 'You have selected ' + $(this).html() + ' ('+$(this).data('value')+').';
		alert(msg);
	}
});
